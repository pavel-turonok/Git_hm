# 1	
Title Регистрация пароля в нижнем регистре	
Description При регистрации пользователя есть возможность регистрации пароля без заглавных букв	
Steps 1. Перейти на форму регистрации	Пользователь незарегистрирован. 
      2. Ввести логин в поле ввода "username"	
			3. Ввести пароль в поле ввода "password" более 8 символов в нижнем регистре			
			4. Тап на кнопку "Register"			
Expected result Появляется сообщение о невалидности введенных данных, либо о необходимости содержания в пароле заглавных букв	Пользователь зарегистрирован. 
Actual result Появляется сообщение "Successfully registered! Please, click Back to return on login page"	
Severity Major
